a2a.client.service_parameters.with_a2a_extensions(extensions: list[str]) -> Callable[[dict[str, str]], None]

   Create a ServiceParametersUpdate that merges A2A extension URIs.

   Unions the supplied URIs with any already present in the A2A-
   Extensions parameter, deduplicating and emitting them in sorted
   order. Repeated calls accumulate rather than overwrite.
